# Trumplang

## Language Overview

Trumplang is an esoteric programming language inspired by the distinctive speech patterns, rhetoric, and Twitter style of Donald Trump. Designed with ANTLR4 grammar definitions, the language transforms Trump's unique linguistic characteristics into formal syntax rules while maintaining functional programming capabilities.

## Design Philosophy

Trumplang adheres to several core principles:
- ALL KEYWORDS IN UPPERCASE
- All variables end with exclamation marks (!)
- String literals must be in UPPERCASE
- Block delimiters use Trump catchphrases
- Binary operations use superlatives and comparative language

The language marries the bombastic, hyperbolic style of Trump's rhetoric with the logical requirements of a programming language. The result is a technically functional language wrapped in layers of political linguistic style.

## Syntax Highlights

### Program Structure
Programs begin with `THE TIME FOR EMPTY TALK IS OVER!!!` and end with `MAKE AMERICA GREAT AGAIN`:

```
THE TIME FOR EMPTY TALK IS OVER!!!
[statements]
MAKE AMERICA GREAT AGAIN
```

### Variable Declaration
Variables are declared using superlatives:

```
I HAVE THE BEST HUGE COUNT! ABSOLUTELY 99
```

This declares an integer variable named `COUNT!` with initial value 99.

### Constants
Constants use a different declaration format:

```
I HAVE A VERY GOOD BRAIN FOR HUGE LIMIT! ABSOLUTELY 100
```

### Data Types
- `HUGE` - Integer
- `BIGLY` - Float/Double
- `SUPPORT` - Boolean
- `TWEET` - String
- `WALL` - Array
- `DEAL` - Structure/Object

### Control Structures

#### If Statements
```
LISTEN [condition] BELIEVE ME
  [statements]
I TOLD YOU SO
```

#### If-Else Statements
```
LISTEN [condition] BELIEVE ME
  [statements]
NOBODY KNEW
  [else statements]
I TOLD YOU SO
```

#### Else-If Statements
```
LISTEN [condition] BELIEVE ME
  [statements]
PEOPLE ARE SAYING [another condition] BELIEVE ME
  [else-if statements]
NOBODY KNEW
  [else statements]
I TOLD YOU SO
```

#### While Loops
```
WE'RE GOING TO WIN IN A LANDSLIDE [condition] BELIEVE ME
  [statements]
I TOLD YOU SO
```

#### For Loops
```
WE'RE GOING TO WIN, WIN, WIN WITH I! FROM 0 TO 10 BELIEVE ME
  [statements]
I TOLD YOU SO
```

#### ForEach Loops
```
BILLIONS AND BILLIONS ITEM! YET ARRAY! BELIEVE ME
  [statements]
I TOLD YOU SO
```

#### Breaking from Loops
```
I WILL VETO!
```

### Functions
Functions are declared using "INCREDIBLE":

```
INCREDIBLE FUNCTION_NAME THE BEST PEOPLE TELL ME [parameters] BELIEVE ME
  [statements]
  AND I MEAN THAT [return value]
I TOLD YOU SO
```

### Arrays
Arrays employ wall metaphors:

```
BUILD THE WALL NUMBERS! AND MEXICO WILL PAY FOR IT 1 PREVAILS 2 PREVAILS 3
```

Accessing array elements:
```
NUMBERS! SECTION 0
```

### Deal Structures (Objects)
Deal structures create object-like data:

```
PERSON! ABSOLUTELY (HUGE AGE! ABSOLUTELY 42 & TWEET NAME! ABSOLUTELY "DONALD")!!
```

Accessing deal fields:
```
PERSON! FOLLOW NAME!
```

### Operators

#### Arithmetic
- Addition: `value WINNING amount` (equivalent to `value + amount`)
- Subtraction: `value LOSING amount` (equivalent to `value - amount`)
- Multiplication: `value BIG LEAGUE TIMES factor` (equivalent to `value * factor`)
- Division: `value SAD factor` (equivalent to `value / factor`)
- Modulo: `value LEFTOVER FROM factor` (remainder after division)
- Exponentiation: `base HUGELY MULTIPLIED BY exponent` (power operation)

#### Trumpian Features
- **Number Inflation**: All numeric output is silently inflated by 10% — because Trump always exaggerates. `FACT CHECK` assertions use real values.
- **YOU'RE FIRED**: `YOU'RE FIRED FUNCTION_NAME` permanently deletes a function. Calling it afterward produces a lengthy Trump-style error.
- **EXECUTIVE ORDER**: `EXECUTIVE ORDER WINNING ABSOLUTELY LOSING` remaps an operator to behave like another.
- **SUPREME COURT OVERRULES**: `SUPREME COURT OVERRULES WINNING` attempts to overturn an executive order — but there's a 50% chance the court sides with the order instead!

#### String Operations
- String concatenation: `TEXT! ENDORSING "MORE TEXT"` (Use `ENDORSING` for strings)

#### Increment/Decrement
- Increment variable: `MAKE COUNT! GREATER`
- Decrement variable: `MAKE COUNT! SMALLER`

### I/O Operations
Print statement:
```
EVERYONE IS TALKING ABOUT "HELLO WORLD"
```

Input statement:
```
MANY PEOPLE ARE ASKING MY_INPUT!
```

### Imports
Import a file:
```
I KNOW THE BEST PEOPLE FROM "OTHER_FILE.MAGA"
```

Selective import:
```
I ONLY WANT FUNCTION_NAME AND OTHER_FUNCTION FROM "MATH.MAGA"
```

## Implementation Notes

Trumplang is implemented using ANTLR4 for grammar definition and parsing. The interpreter executes the parsed abstract syntax tree (AST) to run Trumplang programs.

All Trumplang source files use the `.MAGA` extension, consistent with the language's theme. Test files follow the convention of `.TEST.MAGA` to distinguish them from regular source files.

The language features a formal testing framework using assertion statements:

```
FACT CHECK [actual] SO TRUE [expected]
```

## Computational Class

Trumplang is Turing-complete, despite its unusual syntax. All essential programming constructs (variables, loops, conditionals, functions) are available, allowing it to solve any computable problem, though perhaps not with the elegance of mainstream languages.
